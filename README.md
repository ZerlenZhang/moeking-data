# 萌王数据 (MoeKing Data)

> 🌟 二次元游戏数据分析与社交平台 - 让你的角色养成之路更清晰！

## 项目概述

萌王数据是一个专注于二次元游戏的综合性平台，为玩家提供：
- 📊 游戏数据分析（角色强度、队伍搭配、资源规划）
- 💬 玩家社交（攻略分享、队伍交流、经验交流）
- 🎨 精美二次元UI设计
- 📱 原生Android应用，流畅体验

## 核心功能

### 1. 角色数据库
- 支持原神、崩坏：星穹铁道、绝区零、明日方舟
- 角色属性、技能、弱点分析
- 评级系统（T0-T5）

### 2. 队伍搭配助手
- 智能推荐队伍
- 元素反应分析
- 缺位角色提示

### 3. 社区功能
- 攻略发布
- 队伍交流
- 经验分享

### 4. 数据可视化
- 角色养成进度
- 资源消耗计算
- 概率分析

## 技术栈

- **语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构**: MVVM + Clean Architecture
- **网络**: Retrofit + OkHttp
- **数据**: Room Database
- **图片**: Glide
- **构建工具**: Gradle
